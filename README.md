# Open-Source-Tools-Dashboard
A curated, category-wise dashboard showcasing powerful and popular **open-source tools** used across various technology domains such as Programming, AI/ML, Design, DevOps, Security, and more.

This project is developed as a **team project** with the objective of helping students and developers **discover, understand, and access open-source software** from a single, organized platform.

---

## Project Objectives

- To design a clean and structured web dashboard using **HTML and CSS**
- To categorize open-source tools based on their application domains
- To provide direct access to official **websites and source repositories**
- To promote awareness and adoption of **open-source technologies**
- To work collaboratively as a team on a web development project
